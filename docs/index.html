1. Overview

This is a simple and effective To-Do List or Shopping List application. Users can add, edit, delete, filter items, and persist data using localStorage.

Note: The app is inspired by Brad Traversy's tutorial, with additional features (cancel button, character limit, clear input button, global Escape, etc.) implemented by the developer.

2. Key Features

Add Item

Add new items via button or Enter key

Edit

Update existing items

Delete

Remove single or all items

Filter

Search items (case-insensitive)

Persistence

Data saved in localStorage

Keyboard Support

Enter, Escape, global shortcuts

3. User Scenarios (Usage Flows)

3.1 App Launch

  • All saved items are loaded into the list.
  • If the list is empty:
    • "Clear All" button is hidden.
    • Filter input is hidden.
  • Focus is automatically set to the input field.

3.2 Add New Item

  1. Type text into the input field (e.g., Milk).
  2. Click "Add Item" or press Enter.
  3. Result:
    • Item is added to the list.
    • Input is cleared.
    • Focus remains in the input.
    • "Clear All" and filter field become visible.
Warning: Duplicate items are blocked. Items longer than 15 characters are not allowed.

3.3 Edit Item

  1. Click on an item in the list.
  2. Result:
    • Item is highlighted in gray.
    • Button changes to "Update Item" and turns green.
    • Item text appears in the input field.
    • "Cancel" button appears.
  3. Edit the text and click "Update Item" or press Enter.
  4. Result: Item is updated, edit mode closes.

3.4 Cancel Edit

Click "Cancel" or press Escape anywhere.

Result: Input is cleared, edit mode is canceled.

3.5 Delete Item

  1. Click the red X icon next to the item.
  2. Confirm with "OK" in the dialog.
  3. Result: Item is removed from list and storage.

3.6 Clear All Items

  1. Click "Clear All".
  2. Confirm with "OK".
  3. Result:
    • All items are deleted.
    • Storage is cleared.
    • UI is updated (buttons hidden).

3.7 Filter Items (Search)

  1. Type text into the filter field (e.g., el).
  2. Result:
    • Only matching items are shown.
    • Search is case-insensitive.
  3. To clear filter:
    • Press Escape while in the filter field or
    • Manually clear the input.

3.8 Clear Input (X Button)

When text is in the input, click the X button on the right.

Result: Input is cleared. Edit mode is canceled if active.

3.9 Character Counter

While typing, a counter shows X / 15 below the input.

  • If over 15 characters:
    • Text turns red.
    • Addition is blocked (alert shown).
